Sesión 2


Hyper Text Markup Languaje

Lectura no secuencial

Diferentes vinculos y redes de informacion conectadas entre si.

Una etiqueta puede contener a otra etiqueta.

Identación


Aplicar sucesivos niveles de márgenes de acuerdo a la anidación de etiquetas.
Permite identificar con mayor facilidad problemas de anidación o desbalance en etiquetas de cierre.
Se puede indentar con tabuladores o espacios.

Recomendaciones de escritura

  • Cuida siempre la indentación
  • Prefiere minúsculas para etiquetas y atributos
  • Prefiere comillas dobles para atributos
  • Evita generar líneas demasiado largas
  • Elimina espacios blancos innecesarios

Recomendaciones para nombres de archivos y carpetas

  • Prefiere minúsculas
  • Utiliza sólo letras, números y guiones. Evita usar caracteres inusuales, tildes, símbolos, comillas, etc
  • Utiliza guiones para separar palabras. Evita siempre usar espacios
  • Definir nombres cortos y relevantes, junto con una convención de nombres

Ventajas de una buena semántica


Mejora la accesibilidad del contenido para usuarios que utilizan lectores de pantalla
Puede ser considerado por los motores de búsqueda para influir en el ranking de una página (SEO)
Permite recorrer y rastrear bloques de código significativo
Sugiere el tipo de datos con que debe ser completado

Cascading Style Sheet


Hojas de estilos en cascada

Tipos de escritura


  • Utilizar minúsculas
  • Indentar declaraciones dentro de la regla
  • Sólo un selector por línea
  • Dejar un espacio entre el último selector y la llave de inicio
  • Dejar un espacio entre los dos puntos y el valor

¿Cómo asociar una hoja de estilos a un documento HTML?


---Archivo externo ✅---
  • Se puede insertar tanto en la sección como en (pero se recomienda en )
  • Se puede enlazar a múltiples hojas de estilo
  • Las hojas de estilo se guardan en caché de navegador por lo que optimiza la descarga de la página

--- Etiqueta "Style" ⚠️---
  • Se puede insertar tanto en la sección como en
  • Pueden haber múltiples declaraciones
  • Permite usar todas las posibilidades de CSS
--- Estilos en línea ⚠️ ---
  • Mezcla capa de contenido con capa de presentación
  • Sólo se puede dar estilos a un elemento a la vez
  • Aumentan el tamaño de la descarga del HTML ya que se descargan con éste en cada acceso
  • Al hacer una modificación, se debe replicar en todas las páginas de un sitio
